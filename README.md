```bash

# Access APOLLO at http://localhost:3000
```

## 🔧 **DEVELOPMENT**

### Project Structure
```
├── app/                    # Next.js app directory
│   ├── api/               # API routes
│   ├── components/        # React components
│   ├── lib/              # Core libraries
│   └── types/            # TypeScript types
├── prisma/               # Database schema
├── public/               # Static assets
├── apollo_auto_learning_system.py  # AI learning system
└── apollo_auto_learning_activation.py  # Learning activation
```

### Key Technologies
- **Frontend:** Next.js 14, React 18, TypeScript
- **Backend:** Node.js, Prisma, PostgreSQL
- **AI:** Ollama, Python, Multi-modal models
- **Styling:** Tailwind CSS, Framer Motion
- **Deployment:** Vercel, Docker, Kubernetes

### Development Commands
```bash
# Development
npm run dev              # Start development server
npm run build           # Build for production
npm run start           # Start production server

# Database
npx prisma generate     # Generate Prisma client
npx prisma db push      # Push schema changes
npx prisma studio       # Open database GUI

# AI Learning
python3 apollo_auto_learning_activation.py  # Activate learning
python3 apollo_auto_learning_system.py      # Run learning system

# Testing
npm run test            # Run tests
npm run test:watch      # Watch mode
npm run test:coverage   # Coverage report
```
